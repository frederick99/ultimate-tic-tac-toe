About

Ultimate tic-tac-toe is a modified version of 3*3 Tic tac toe board game. It is composed of nine independent tic-tac-toe boards arranged in a 3-by-3 grid.

How to play

Definitions

Local board
each small 3-by-3 tic-tac-toe board
Global board
the larger 3-by-3 board

Rules

  • The game starts with X playing wherever they want in any of the 81 empty spots.
  • This move 'sends' their opponent to its relative location on the Global Board as per the previous move.
  • The game proceeds till one of the players has captured 3 local boards in a row (up, down, across, or diagonally). This player is declared as the winner.
  • When all 9 local boards are captured, the game is over. If no player has captured 3 local boards in a row, the game ends in a tie.
